
# ISIMM Archive Backend

Strapi CMS backend for the ISIMM University academic archive system. Manages PDF uploads, corrections, programs, and levels.

## 🚀 Features
- Strapi CMS content management
- PDF document storage & retrieval
- Program and level management
- Secure
- API endpoints for frontend integration

## 📦 Deployment
- **Deployed on**: Strapi Cloud
- **Connection**: Connected to ISIMM Archive Frontend (Netlify)

## 📝 License
Only **License INFO1** is currently available (testing phase). Not all levels/programs are implemented yet.

## 🛠 Tech Stack
- Strapi CMS
- Strapi Cloud (deployment)
